레이아웃(layout)

        사전적 의미 : 배치, 정리
        기술적 의미 : 구성 요소를 제한된 공간에 효율적으로 배치하는 것

화면 배치(표시, 영역) 지정 : display 속성

        요소가 화면에 어떻게 보여질지 형식을 지정하는 속성

        - block : 화면을 한 줄 다 차지하는 수직 분할
                + width, height 지정 가능
        
        - inline : 화면에서 필요한 만큼 행을 가짐 글자나 문자영 작동 수평 분할
                + width, height 지정 불가능

        - inline-block : inline과 block 장점을 활용
                        inline처럼 필요한만큼 행을 지님 width, height 지정 가능
                        + img, iframe처럼 다른 소요도 사용가능
        - none : 요소는 존재하나 화면에는 보이지 않게 함 (투명 태그)

        - flex : 요소가 정렬되는 방향, 요소간의 간격을 유연하게 처리하는 방식

        -grid : 격자무늬 형식으로 화면을 쉽게 분할하는 형식
