background 속성들 완벽 비교

        핵심 차이점
        background-color : 단색만 가능
        background-image : 이미지, 그라디언트 등 복잡한 배경이 가능
        background       : 모든 배경 속성을 한번에 설정하는 통합 속성 
                            단색 가능, 이미지 가능, 그라디언트 조합 가능

background-color (배경색)

용도 : 단순한 단색 배경을 설정할 때 사용

        /*기본 문법*/
        background-color : 색상값;

        background-color : #ff6b6b; /* 16진수로 표현하는 색상 표*/
        background-color : red; /* 색상 명칭으로 표현하는 색상 표*/
        background-color : rgb(255, 0, 0); /* rgb or rgba로 표현하는 색상 표*/

background-image (배경이미지, 그라디언트)

용도 : 이미지, 그라디언트, 패턴 등 복잡한 배경을 설정할 때 사용 

        /*기본 문법*/
        background-image : 색상값;

        background-image : url('이미지가 들어있는 폴더위치/이미지.이미지확장자'); 
        /* 
        이미지가 들어있는 폴더로 접속하여 이미지를 가져오거나,
        이미지 주소값을 인터넷에서 가져와 작성할 수 있음
        */
        background-image : linear-gradient(각도, 색상1, 색상2); /* 선형 그라디언트 */
        background-image : radial-gradient(각도, 색상1, 색상2); /* 원형 그라디언트 */
단색 빨간 배경
단색 초록 배경
단색 파란 배경

background-image (배경이미지, 그라디언트)

용도 : 이미지, 그라디언트, 패턴 등 복잡한 배경을 설정할 때 사용 

        /*기본 문법*/
        background-image : 색상값;

        background-image : url('이미지가 들어있는 폴더위치/이미지.이미지확장자'); 
        /* 
        이미지가 들어있는 폴더로 접속하여 이미지를 가져오거나,
        이미지 주소값을 인터넷에서 가져와 작성할 수 있음
        */
        background-image : linear-gradient(각도, 색상1, 색상2); /* 선형 그라디언트 */
        background-image : radial-gradient(각도, 색상1, 색상2); /* 원형 그라디언트 */
선형 그라디언트
원형 그라디언트

background (통합 속성)

용도 : 여러 배경 속성을 한 번에 설정할 때 사용 (color + image + position + repeat 등) 

        /* 기본 문법*/
    background : [color] [image] [repeat] [position] [size];
                 color나 image 둘 중 하나는 필수로 들어가야 하지만 나머지는 필수가 아님
    /* 예제 */
    background : #ff6b6b; /* 단색만 */
    background : url('image.png') no-repeat center; /* 이미지 + 옵션 */
    background : linear-gradient (각도, 색상1 #fecdc4, 색상2 #4ecdc4);
색상 + 그라디언트 조합
색상 + 패턴 조합
복잡한 다중 그라디언트 조합
