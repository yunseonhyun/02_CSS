일반 구조(위치) 선택자

        형제 관계에 있는 요수 중에서 요소만을 선택하는 선택자

        * 중요 * -> 위치를 기준으로 구분함

        (A : 선택된 요소)
        A:first-child -> 형제들 중 첫 번째가 A가 맞다면 선택

        A:last-child -> 형제들 중 마시막에 A가 맞다면 선택

        A:nth-child(숫자 또는 수역) -> 형제들 중 n번째가 A가 맞다면 

        A:nth-last-child(숫자 또는 수열) -> 형제들 중 뒤에서부터 n번째가 A가 맞다면
