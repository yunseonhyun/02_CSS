폰트 어썸

        react와 같은 라이브러리, 프레임워크 (기능 모음)을 이용하면 
        간편하게 사용할 수 있지만
        현재 순수 html을 사용하고 있기 때문에
        CDN이나 파일을 가져오는 형식을 사용해서
        외부에서 만들어진 코드를 사용할 수 있음

        1번방법 CDN(Content Delivery Network)
        파일을 저장해서 불러오는 것이 아니라 웹 사이트에서 필요한 기능이나 속성을 가져오는 방법
        
        
        장점 : 항상 최신 아이콘을 사용할수 있음
                주소만 작성하면 되기 때문에 설정 간편
                개발자가 원하는 필요한 스타일이나 아이콘 불러올 수 있음
        단점 : 인터넷이 끊길 경우 작동 x
                외부 인터넷 사이트로 접속해서 개발자가 원하는 속성이나 기능을
                가져오기 때문에 속도에 미세한 영향을 줄 수 있음

        2번방법 Local File(개발자의 컴퓨터에 저장된 기능이나 속성을 가져오는 방법)
        

        장점 : 인터넷이 끊긴 환경에서도 기능을 사용할 수 있음
                외부 인터넷 사이트로 접속해서 가져오는 것이 아니기 때문에
                상대적으로 속도가 빠를 수 있음
        단점 : 수동으로 업데이트
                설정이 복잡할 수 있고, 아이콘 종류나 기능에 제한이 있을 수 있음
