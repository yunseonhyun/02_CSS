자식 선택자 ( > )

        지정된 요소 바로 한 단계(1 level/depth) 하위 요소 선택

        작성법
        
            선택자1 > 선택자2{
                CSS 코드;
            }

            - 선택자 1 : 부모 요소(감싸고 있는 요소)
            - 선택자 2 : 자식 요소(하위 요소)
  • 자식
  • 자식2
  • 자식3
  • 자식4

후손 선택자(작성법 : 띄어쓰기)

        지정된 요소의 모든 하위 요소 선택 (n level 이하 모두)

        작성법
        
            선택자1 선택자2 {
                CSS 코드;
            }

            선택자1 : 부모(조상) 요소의
            선택자2 : 후손 요소

반응 선택자

        사용자의 움직임에 반응하여 스타일이 변경되게 하는 선택자1

        선택자:active -> 선택된 요소가 클릭 되고 있을 때
        
        선택자:hover -> 선택된 요소 위에 마우스가 올라갔을 때
:active
:hover

상태 선택자

        입력 양식(input, input 관련 태그)의 상태에 따라 선택되는 선택자

        :focus      -> 요소에 초점이 맞춰졌을 때
                        (input 요소를 클릭해서 입력할 준비가 되었을 때)
                
        :checked    -> 요소가 체크 되었을 때 (redio, checkbox)

        :enabled / :disabled    -> 요소가 활성/ 비활성화 되었을때
:focus
아이디 입력 :
:checked
:enabled / disabled
