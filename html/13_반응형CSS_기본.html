기본 반응형 CSS

1. 미디어 쿼리란?

미디어 쿼리(Media Query)는 CSS3에서 도입된 기능으로, 기기의 화면 크기, 해상도, 방향 등에 따라 다른 스타일을 적용할 수 있게 해주는 CSS 기법 

@media(조건) {
    /* 조건이 맞을 때 적용할 CSS 스타일 */
}

2. 기본문법

2. 1최대 너비 조건

@media (max-width : 768px) {
    /* 화면 너비가 768px 이하일때 적용 */
    .container {
        width: 100%;    -> 기기 너비(가로)의 크기가 768px이하이면 
                            기기의 크기에 맞춰서 div 태그들의 요소 크기를
                            조정하겠다
        padding: 10px;
    }
}

2.2 최소 너비 조건

@media(min-width:769px) {
    /* 화면 너비가 769px 이상일 때 적용 */
    .container {
        width:100px;
        margin: 0 auto;
    }
}

2.3 범위 조건

@media(min-width: 768px) and (max-width: 1024px) {
    /*화면 너비가 768px ~ 1024px 사이일 때 적용*/
    .container {
        width: 90%;
    }
}

3. 주요 브레이크 포인트

기기 유형 화면 크기 미이더 쿼리 예시
모바일 (세로) 0~480px @media(max-width: 480px)
모바일 (가로) 481px~768px @media(max-width: 768px)
태블릿 769px~1024px @media(max-width: 1024px)
데스크톱 1025px~1200px @media(max-width: 1200px)
대형화면 1201px~ @media(min-width: 1201px)
