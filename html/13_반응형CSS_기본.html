기본 반응형 CSS

1. 미디어 쿼리란 ?

미디어 쿼리(Media Query)는 CSS3에서 도입된 기능으로, 기기의 화면 크기, 해상도, 방향 등에 따라 다른 스타일을 적용할 수 있게 해주는 CSS 기법 

@media(조건) {
    /* 조건이 맞을 때 적용할 CSS 스타일 */
}

2. 기본문법

2. 1 최대 너비 조건

@media (max-width : 768px) {
    /* 화면 너비가 768px 이하일 때 적용 */
    .container {
        width: 100%;  -> 기기 너비(가로)의 크기가 768px 이하이면
                            기기의 크기에 맞춰서 div 태그들의 요소 크기를
                            조정하겠다.
        padding : 10px;
    }
}

2.2 최소 너비 조건

@media(min-width:769px) {
    /* 화면 너비가 769px 이상일 때 적용 */
    .container {
        width:1200px;
        margin: 0 auto
    }
}

2.3 범위 조건

@media(min-width: 768px) and (max-width: 1024px) {
    /* 화면 너비가 768px ~ 1024px 사이일 때 적용*/
    .container {
        width: 90%;
    }
}

3. 주요 브레이크 포인트

기기 유형 화면 크기 미디어 쿼리 예시
모바일 (세로) 0~480px @media(max-width: 480px)
모바일 (가로) 481px~768px @media(max-width: 768px)
태블릿 769px~1024px @media(max-width: 1024px)
데스크톱 1025px ~ 1200px @media(max-width: 1200px)
대형화면 1201px ~ @media(min-width: 1201px)

4. 실제예시

💡 팁: 현재 화면 크기를 변경해보세요! 아래 박스의 색상과 텍스트가 변경됩니다.
반응형 데모 박스

4. 1 완전한 반응형 예시

/* 기본 스타일 (데스크톱) */
.container {
    width:1200px;
    margin : 0 aut0;
    padding : 20px;
}

/* 태블릿 (768px 이하) */
@media (max-width:768px) {
    .container {
        width: 100%;
        padding : 15px;
    }
}
/* 모바일 (480px 이하) */
@media (max-width:480px) {
    .container {
        padding : 10px;
    }
}

5. 주의사항

5.1 뷰포트 메타 태그

 참고로 꺽쇠 태그는 pre 내부에 존재하게 되면
 태그 형태로 실행
 < → <    less then     , 작다
 > → >    greater then  , 크다
 & → &   ampersand     , 엠퍼샌드
 " → "  quotation mark, 따옴표
 ' → '  apostrophe    , 아포스트로피
 <meta name="viewport" content="width=device-width, initial-scale=1.0" />

모바일 기기에서 올바른 반응형 동작을 위해 반드시 HTML head 에 포함해야합니다.

5.2 모바일 퍼스트 vs 데스크톱 퍼스트

모바일 퍼스트 (권장)

/* 모바일 기본 스타일 */
.container { width: 100%; }

/* 태블릿 이상 */
@media (min-width: 768px) {
    .container { width: 750px; }
}

/* 데스크톱 이상 */
@media (min-width: 1024px) {
    .container { width: 1200px; }
}

5.3 자주 사용하는 속성들

@media (max-width: 768px) {
    /* 폰트 크기 조정 */
    h1 { font-size: 24px; }
    
    /* 여백 조정 */
    .container { padding: 10px; }
    
    /* 너비 조정 */
    .menu-item { width: 100%; }
    
    /* 숨기기/보이기 */
    .desktop-only { display: none; }
    
    /* 플렉스 방향 변경 */
    .flex-container { flex-direction: column; }
}

6. 실전 팁

  • 점진적 향상 : 모바일로부터 시작해서 큰 화면으로 확장
  • 터치 친화적 : 버튼 크기 최소 44px 이상 유지
  • 읽기 쉬운 텍스트 : 모바일에서 최소 16px 폰트 크기
  • 성능 고려 : 불필요한 이미지나 스타일은 해당 화면에서만 로드
